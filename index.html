  • 전국 교통사고 다발 지역 시각화 - 양지수

    전국 교통사고 다발지역(2012-2021, 2017-2023) 데이터 중에서 최근 10년치 데이터만 가져와 위치코드가 같은 데이터끼리 그룹화하는 작업을 진행함. 이후 이 위치코드를 이용하여 geojson과 매핑, 사고건수를 지도상의 색의 진하기로 표현함.
    각각 시도별 데이터를 추출하여, 사용자가 selectbox로 시도명을 선택하면 똑같은 기능 실행할 수 있도록 함.

  • 운전하기에 적절한 시간대 시각화 - 선소휘

    한국도로교통공단 교통사고분석시스템 koroad에서 제공하는 경찰 db 시간대별 교통사고 데이터를 이용함. 필요한 변수를 선별하고 시군구별 geojson 지역명칭과 잘 매핑되도록 전처리 하여 시각화함.

  • 기온과 교통사고 - 선소휘

    한국도로교통공단 교통사고 데이터와 국가 통계 포털 kosis의 평균최고기온 데이터를 이용함. 각 데이터의 월별 통계를 계산하는 전처리 과정을 통해 시각화 함.

  • 특정 기간의 교통사고 추이 시각화 - 양지수

    특정 기간의 교통사고 추이에 대해 세운 가설을 바탕으로, 전체 데이터에서 특정 기간만 추출하여 그래프를 그려 시각화하여 가설에 대한 결론을 내릴 수 있도록 함.

  • 미세먼지와 교통사고 - 김재현

    6년치 도로교통공단 교통사고 건수, 에어코리아 미세먼지 측정자료 취합 및 분석,
    시계열 정보 60일 Moving Average 적용

  • 주식 등락률과 교통사고 - 박주홍

    주가 변동에 따른 교통사고 변화
